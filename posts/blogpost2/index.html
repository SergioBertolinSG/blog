    How to use several git accounts in same computer

    Sometimes we have the need of using several git accounts. We can be doing different projects, working for different companies, maybe using different collaborative version control tools and other reasons.

    This leads to different problems. For example you make commits with the wrong account and forces you to amend them or you cannot interact with the tools because it needs different authentication.

    To solve this we can make use of the git config possibilities.

    First we need different public keys for each of the tools we want to use with different accounts.

    When generating the keys is important to give them a descriptive name.

    Lets imagine we have three tools we want to interact, for example github, gitlab selfhosted and selfhosted bitbucket.

    We create the public keys like

    ~/.ssh/id_github for github

    ~/.ssh/id_gitlab for selfhosted gitlab

    ~/.ssh/id_goodcorp for selfhosted bitbucket

    (creating the keys can be done using ssh-keygen -t ed25519 or ssh-keygen -t rsa)

    Then create a git config file in .ssh named config (or edit the already created one).

    And put the hosts

    ~/.ssh/config

    Host github.com

  User git

  IdentityFile ~/.ssh/id_github
 

Host gitlab.yourhost.es

  User git

  IdentityFile ~/.ssh/id_gitlab

 
Host bitbucket.goodcorp.com

  User git

  IdentityFile ~/.ssh/id_goodcorp

    In the respective tools you will have to associate your public keys to your accounts.

    To make git know the right email and name to use in your commits you will need two extra steps.

    Edit the file ~/.gitconfig to set the right file to use depending on the folder.

    Lets imagine we have a default name and email we want to use in general. And a particular git repository in ~/proyecto_personal where we want to use a different user.

    ~/.gitconfig

    [user]

        name = John Doe
        email = John.Doe@example.com

[core]

        editor = vi

 

[includeIf "gitdir:~/proyecto_personal/"]

        path = ~/proyecto_personal/.gitconfig

    In the git repository we need to create that .gitconfig file and edit the details.

    ~/proyecto_personal/.gitconfig

    [user]

        name = Jane Doe
        email = Jane.Doe@example.com
